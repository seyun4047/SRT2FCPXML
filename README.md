# SRT2FCPXML
SRT to FCPXML converter
- Convert your SRT sub to Finalcut pro title!

## Convert with basic title style
```bash
# Execute
python3 srt2fcpxml.py srt_filename.srt fcpxml_filename.fcpxml 60 1
```

# If you want to make your title on the box !
```bash
# move your directory and change the path
# mutzin.motiv & mutzin_title.fcpbundle

# Execute 
python3 srt2fcpxml.py srt_filename.srt fcpxml_filename.fcpxml 60 1
```
